# wp-guardian
Wordpress vulnerability scanner and reporting tool. 

## Installation

Clone github repo

    git clone https://github.com/ricardolopezg/wp-guardian


Install dependencies

    pip install -r requirements.txt


Usage

    python3 wp_guardian.py http(s)://www.example.com/
